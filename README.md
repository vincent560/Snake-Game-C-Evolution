# 經典貪食蛇遊戲：從邏輯實作到系統優化 (Greedy Snake 3.0)

本專案為大學期末專題作品，旨在透過開發經典遊戲「貪食蛇」，實踐 C 語言底層記憶體管理與視窗系統互動。

## 🚀 核心技術亮點

- **動態蛇身管理 (Linked List)**：捨棄傳統陣列，採用指標與鏈結串列 (Single Linked List) 維護蛇身。這使得遊戲在蛇身成長時能高效配置記憶體，並精準控制每一節的位置更新。
- **系統層級問題修復 (Windows API & IME)**：
    - **IME 衝突解決**：針對終端機遊戲常見的輸入法干擾，透過 `imm.h` 強制切換為英文輸入模式，確保玩家操作流暢不卡頓。
    - **游標渲染優化**：實作 `gotoxy()` 函式配合 `CONSOLE_CURSOR_INFO` 隱藏游標，解決了傳統螢幕閃爍問題，大幅提升視覺體驗。
- **資料持久化**：實作檔案 I/O 功能 (`score.txt`)，自動記錄玩家最高得分，具備基礎的遊戲存檔概念。

## 🛠 實作功能
1. **地圖繪製**：20x20 符號邊框偵測與渲染。
2. **成長機制**：偵測碰撞食物後，動態新增 Node 並更新總分。
3. **碰撞偵測**：包含邊界碰撞與蛇身自我碰撞判定 (`gameOver` 邏輯)。
4. **重置系統**：具備一鍵重啟遊戲流程 (`restartGame`) 與完整的記憶體回收機制 (`freeSnake`)。
